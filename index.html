Bezier Curve Playground

This is a Bezier Curve playground, tap any places in white area to create one control points.
Technically, the generated curve is the polygon based on the number of points, more points means the curve will be more smoother, default points number is 1000
Then click APPLY button to generate Bezeir Curve. Click CLEAR button to clear the canvas.
